# Lab 4
## Linux CLI ( Command Line Interface )

### Shell command

- pwd

shows the current path in a hierarchical directory
```bash
$ pwd
/home/<username>
```

- cd

change directory
```
$ cd Desktop
~/Desktop $
```

- ls

list files and directories
```
$ ls
<file 1> <file 2> <file 3> <folder 1> <folder 2> <folder 3>
```

- arguments
  
* /   
root

* .
current directory

* ..
upper-level directory

* ~
home of current user

* /
absolute path

* ./
relative path

* ../
relative path
